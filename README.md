## 📖 Overview

<a href="#pdfSqueezer">PDF Squeezer</a> serves anyone who regularly handles PDF files that are too large for practical use — the scanned documents that email servers reject for size, the design portfolio PDFs that take minutes to upload, the client reports that fill storage quotas — the routine PDF size problem that knowledge workers encounter regularly.

The <a href="#pdfSqueezer">office and administrative worker use case</a> is most common — processing scanned documents that come from office scanners at unnecessarily high resolution, compressing incoming PDFs for archiving, and preparing documents for email delivery when file size limits are a factor. PDF Squeezer addresses the daily reality that PDF files are often much larger than they need to be for their intended use. The <a href="#pdfSqueezer">designer and creative professional workflow</a> uses PDF Squeezer to reduce portfolio PDFs, design presentation files, and client deliverable PDFs to sizes practical for web delivery and email — design files often embed high-resolution images that are appropriate for print but excessive for digital sharing.

The <a href="#pdfSqueezer">Mac App Store one-time purchase</a> provides permanent access without subscription — a utility that pays for itself the first time it prevents a large PDF from bouncing from an email server or timing out during upload. <a href="#pdfSqueezer">Simple interface without technical knowledge</a> required — drag files in, adjust quality if needed, compress. PDF optimization knowledge is built into the application's default settings.

<a href="#pdfSqueezer">Before and after file size display</a> shows the original and compressed sizes for each processed file — confirming the compression achieved and the ratio of size reduction for each document. <a href="#pdfSqueezer">Output folder configuration</a> sends compressed PDFs to a specified location rather than overwriting originals — maintaining the original files while placing compressed versions in a designated output directory.

<a href="#pdfSqueezer">Processing speed</a> — PDF Squeezer processes files quickly compared to manual optimization workflows, making batch processing of large collections practical within a reasonable time. <a href="#pdfSqueezer">macOS integration</a> includes Quick Action support for compressing PDFs directly from Finder's right-click menu without opening PDF Squeezer's main interface — accessible compression from wherever PDFs are encountered in the filesystem.
